# Color Mapping Guide

## Old Colors → New Colors

### Background Colors
- `#F8F3D9` (Light Beige BG) → `#EBF4F6` (Very Light Teal)
- `#EBE5C2` (Beige Accent) → `#7AB2B2` (Soft Teal)

### Primary/Button Colors  
- `#B9B28A` (Tan Buttons) → `#088395` (Teal Blue)
- `#504B38` (Dark Brown Text/Borders) → `#09637E` (Deep Teal)

### Usage Pattern:
- Main page background: `#EBF4F6`
- Navbar/Sidebar background: `#7AB2B2`
- Primary buttons: `#088395`
- Button hover: `#09637E`
- Text/borders: `#09637E` or `#1C1C1C`
- Card backgrounds: white with `#09637E` borders
- Accent backgrounds: `#EBF4F6`
