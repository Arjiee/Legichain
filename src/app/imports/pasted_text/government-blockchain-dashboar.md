# Figma AI Prompt – Upgrade Dashboard to Government Blockchain Explorer Style

**Prompt:**

Update the **GMA Cavite Blockchain Barangay Ordinance Monitoring System** dashboard to look like a **modern government blockchain monitoring platform**. The design must be professional, clean, and easy to understand for barangay officials and citizens.

---

# 1. Government Dashboard Layout

Create a **3-section dashboard layout**:

### Top Section – System Statistics Cards

Add four statistic cards that update in real time.

Cards should show:

**Published Ordinances**
Total number of ordinance documents uploaded.

**Verified on Blockchain**
Total number of ordinance or violation records verified on the blockchain ledger.

**Active Ordinances**
Total number of currently implemented ordinances across all barangays.

**Total Violations Recorded**
Total number of people who violated ordinances across all barangays.

Each card should display:

* Large number counter
* Small description text
* Small status indicator icon

---

# 2. Ordinance Monitoring Table

Add a **large data table** in the center of the dashboard.

Table columns:

* Ordinance ID
* Barangay
* Ordinance Title
* Category
* Date Implemented
* Violations This Year
* Total Violations
* Enforcement Count
* Blockchain Status
* View Details

Add functions:

* Search bar
* Barangay filter
* Ordinance category filter
* Pagination for large records

---

# 3. Blockchain Transaction Explorer Panel

Create a **Blockchain Explorer panel** similar to real blockchain explorers.

Show the latest violation records stored in the blockchain.

Columns:

* Record ID
* Barangay
* Ordinance Violated
* Transaction Hash (TxHash)
* Block Number
* Timestamp
* Verification Status

Add a **View Blockchain Details button** for each record.

When clicked, open a **Blockchain Transaction Details page**.
