Prompt Title:

Enhance Barangay Projects Module with Detailed Information and Blockchain-Based Transparency

Prompt:

Update the LegiChain: Blockchain-Powered Barangay Project Monitoring and Transparency System by enhancing the Barangay Projects module. The system should provide comprehensive project details, financial transparency, and blockchain-based verification to ensure that all project information is accurate, secure, and trustworthy for both administrators and citizens.

✅ 1. Rename and Standardize Modules
Ensure that all system references use the following terminology:
“Ordinance Records” → “Project Records”
“Ordinance Monitoring Dashboard” → “Project Monitoring Dashboard”
“Ordinance Details” → “Project Details”
Apply these changes consistently across the Admin Panel and Citizen Dashboard.
✅ 2. Barangay Scope

Limit all project records to the following barangays:

Poblacion 1
Poblacion 2
Poblacion 3
Poblacion 4
Poblacion 5

Ensure that all dropdowns, filters, and database validations only allow these barangays.

✅ 3. Project Categories

Organize projects into the following categories for better classification and filtering:

Infrastructure & Physical Improvement
Health, Sanitation, and Environment
Safety, Order, and Social Services
Livelihood, Education, and Agriculture

Each project should be tagged with one of these categories.

✅ 4. Detailed Project Information
Add a “Project Details” Page

When users select “View Details” for a project, display the following structured information:

A. Basic Project Information
Field	Description
Project ID	Unique identifier
Project Title	Name of the project
Barangay	Poblacion 1–5
Category	Project classification
Description	Detailed explanation of the project
Location	Specific area within the barangay
Start Date	Project commencement date
Expected Completion Date	Planned completion
Actual Completion Date	If completed
Project Status	Planned, Ongoing, Completed, or Cancelled
Implementing Office	Responsible barangay unit
Beneficiaries	Target community or sector
B. Project Funding & Utilization

Include a dedicated section to ensure financial transparency:

Field	Description
Total Approved Budget	Allocated funds
Funding Source	Barangay Fund, LGU, National Government, NGO, etc.
Amount Released	Funds disbursed
Amount Utilized	Actual expenses
Remaining Balance	Auto-calculated
Utilization Status	Not Started, Partially Utilized, Fully Utilized
Proof of Expenditure	Receipts, invoices, liquidation reports
Last Updated	Timestamp of latest financial update
C. Blockchain Verification

To ensure immutability and trust, integrate blockchain technology for each project.

Blockchain Data to Display
Transaction Hash (TxHash)
Block Number
Timestamp
From Address (Barangay/Admin)
To Address (Contractor or Public Record)
Document Hash (SHA-256)
Verification Status: Verified on Chain
User Interface Features
Add a “View Blockchain Proof” button.
Display blockchain information in a modal or separate tab.
Record every major update (creation, funding, completion) as a blockchain transaction.
